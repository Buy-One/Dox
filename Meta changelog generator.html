REAPER Meta Changelog Formatter

The changelog can be obtained from A) reaper.fm/whatsnew.txt B) whatsnew.txt file which comes with the REAPER install, or C) called within REAPER via Help -> Check for new versions of REAPER.
Note: The formatted changelog needs proofing due to possible duplicates in different meta categories and appearence of non-categorized change descriptions in the meta category "OTHER", which nevertheless still may fit any of the meaningful meta categories. In these cases refer to the Keyword Classificaton Legend below to determine whether the change description fits the meta category it has been placed under.
